# NoCavesNoRavines
Allows disabling caves and ravines in Minecraft

#### Versions
* Rift 1.13.2 (Incomplete)
* Fabric 1.14.4 (Complete with Config Screen)
* Fabric 1.15.2 (Complete with Config Screen)
* Fabric 1.16.1 (Complete with Config Screen)
* Forge 1.14.4 (Complete with Config file)
* Forge 1.15.2 (Complete with Config file)
* Forge 1.16.1 (Complete with Config file)
