# NoCavesNoRavines
Allows disabling caves and ravines in Minecraft

#### Versions
* Rift 1.13.2 (Incomplete)
* Fabric 1.14.4 (Complete and Has Config Screen)
* Fabric 1.15.2 (Complete and Has Config Screen)
* Fabric 1.16.1 (Complete and Has Config Screen)
* Forge 1.14.4 (Complete and Has Config file only)
* Forge 1.15.2 (Complete and Has Config file only)
