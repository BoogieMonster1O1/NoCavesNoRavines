# NoCavesNoRavines
Allows disabling caves and ravines in Minecraft

#### Versions
* Rift 1.13.2
* Fabric 1.14.4
* Fabric 1.15.2
* Fabric 1.16-rc1 (Requires Fabric loader 8.8)
* Forge 1.15.2
